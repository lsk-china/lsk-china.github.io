20.1 磁现象 磁场

Posted on 2023-04-01 In Notes

20.1 磁现象 磁场

磁性

  • 磁性：能吸引**铁、钴、镍**等物质的性质

  • 磁体：具有磁性的物体

  • 磁极

    • 定义：磁体上磁性最强的部分叫磁极
    • 指向性：小磁针静止时指南的磁极叫南极(S)，指北的磁极叫北极(N)
    • 作用规律：同名磁极相互排斥，异名磁极相互吸引

  • 磁化

    • 定义：一些物体在磁体或电流的作用下获得磁性，这种现象叫做磁化

    • 磁化的两种方法

      • 通过磁体磁化

        与磁体接触或靠近：磁铁靠近铁钉，铁钉被磁化后，铁钉与磁体接触的部分间形成异名磁极，异名磁极相互吸引

      • 通过电流磁化

  • 去磁的方法有：加热、敲击、摩擦

20.2 电生磁

Posted on 2023-04-01 In Notes

20.2 电生磁

电流的磁效应

  • 电流的磁场：通电导线周围存在与电流方向有关的磁场，这种现象叫做电流的磁效应

  • 奥斯特实验

    现象：

    • 当之导线接触电池通电时小磁针发生偏转
    • 断电时，小磁针又回到原来的位置
    • 当改变直导线中电流方向时，小磁针偏转方向也发生变化

    结论：

    • 通电导线周围存在磁场
    • 通电导线周围磁场方向与电流方向有关

    意义：证实了电流周围存在磁场，是第一个揭示了电荷磁之间的联系的人

通电螺线管的磁场

  • 螺线管：将导线绕在圆筒上做成螺线管，通电后各圈导线磁场产生叠加，磁场增强

  • 结论

    • 通电螺线管外部磁场和条形磁体的磁场一样
    • 通电螺线管两端的极性与电流方向有关

安培定则

  • 内容：用右手握住螺线管，让四指指向螺线管中的电流方向，则拇指所指的那段就是螺旋管的N极

  • 用途：判定通电螺线管的极性与螺线管中电流方向之间的关系

20.3 电磁铁 电磁继电器

Posted on 2023-04-01 In Notes

20.3 电磁铁 电磁继电器

电磁铁

  • 定义：插入铁芯的通电螺线管称为电磁铁
  • 工作原理：
    • 电流的磁效应
    • 磁化

探究电磁铁的性质

  • 实验目的
    • 探究磁性的有无
    • 探究磁性的的强弱和哪些因素有关
    • 探究电磁铁磁极与哪些因素有关

实验一：探究磁性的有无

  • 现象：通电时吸引铁钉，断电时铁钉掉下来
  • 结论：通电有磁性，断电没有磁性
  • 研究方法：转换法

实验二：探究磁性的强弱与哪些因素有关

  • 研究方法

    • 转换法：根据吸引铁钉的多少判断电磁铁的磁性强弱
    • 控制变量法：探究电磁铁磁性强弱与哪些因素有关

  • 结论

    • 匝数一定时，通入电流越大，电磁铁的磁性越强

    • 电流一定时，线圈匝数越多电磁铁的磁性越强

实验三：探究电磁铁磁极与什么因素有关

  • 研究方法：转换法
  • 结论：电磁铁的极性与电流方向有关

三、电磁铁的有点

  • 磁性有无由通断电来控制

  • 磁极由电流方向来控制

  • 磁性强弱由电流大小、线圈匝数、线圈形状来控制

  • 注意

    • 铁芯用软铁

四、电磁铁的应用

电磁起重机，电铃

五、电磁继电器

  • 电磁继电器实质：利用电磁铁来控制工作电路的一种开关

  • 工作原理：利用电流的磁效应

  • 应用

    • 空气开关

    • 水位自动报警器

8.3 金属的保护和利用

8.3 金属的保护和利用

铁的冶炼

  • 铁矿石： 赤铁矿 $\ce{Fe2O3}$、磁铁矿$\ce{Fe3O4}$、菱铁矿$\ce{FeCO3}$

工业炼铁

  • 原料：铁矿石、焦炭、石灰石、（空气）

  • 铁矿石选磁铁矿，赤铁矿是因为

    • 含铁量高
    • 产生污染少（有害元素少）

  • 焦炭的作用

    • 提高炉温

    • 提供还原剂 $\ce{CO}$

  • 石灰石作用

    • 将矿石中的 $\ce{SiO2}$ 转变为炉渣

  • 设备：高炉

  • 原理

    • 在高温条件下用 $\ce{CO}$ 把 $\ce{Fe}$ 从铁矿石中还原出来
    • $\ce{Fe2O3 + 3CO ->[高温] 2Fe + 3CO2}$ (赤铁矿)
    • $\ce{Fe3O4 + 4CO ->[高温] 3Fe + 4CO2}$ (磁铁矿)

实验室炼铁

  • $\ce{Fe2O3 + 3CO ->[高温] 2Fe + 3CO2}$

  • 现象：红色固体变成黑色固体，澄清石灰水变浑浊

  • 实验开始要先通 $\ce{CO}$ 后点燃酒精喷灯目的：排尽空气，防止爆炸

  • 结束后要先停止加热，冷却后再停止通 $\ce{CO}$ 目的是：防止热的铁再次被氧化，防止石灰水倒吸

  • 酒精灯的作用：处理尾气，防止污染空气

  • 石灰水作用：检验并吸收 $\ce{CO2}$

  • 与工业炼得的铁有何不同？工业上是生铁，是混合物；实验室是纯铁，是纯净物

金属资源的保护

铁生锈条件

  • 铁生锈的条件：铁与水和氧气同时接触（在潮湿的空气中）

  • 注意

    • 煮沸并迅速冷却的蒸馏水目的：除去水中的 $\ce{O2}$
    • 植物油作用：隔绝 $\ce{O2}$

  • 铁锈的主要成分：红色的 $\ce{Fe2O3}$

  • 除锈方法

    • 用砂纸打磨

    • 用酸洗

      $\ce{Fe2O3 + 6HCl -> 3H2O + 2FeCl3}$

      $\ce{Fe2O3 + 3H2SO4 -> 3H2O + Fe2(SO4)3}$

      注意：酸不能过量，浸泡时间不能过长

  • 铁生锈好处：做食品抗氧化剂（双吸剂），做热贴

防止铁生锈

  • 原理
    • 隔绝 $\ce{O2}$
    • 隔绝水
  • 方法
    • 保持铁制品表面的干燥洁净
    • 在铁制品表面形成保护膜，如刷漆、涂油、电镀、烤蓝
    • 制成合金

保护金属资源途径

  • 防止金属锈蚀

  • 金属回收利用

    优点：节约金属资源，减少环境污染

  • 寻找金属的代用品

  • 有计划合理的开采矿物

8.2 金属的化学性质

Posted on 2023-04-01 In Notes

8.2 金属的化学性质

一、能与氧气反应

  • 常温下

    $\ce{ 4Al + 3O2 -> 2Al2O3}$

    铝制品耐腐蚀的原因：常温下铝与氧气反应，生成一层致密的氧化铝薄膜，从而阻止铝进一步氧化

  • 加热，高温

    $\ce{2Cu + O2 ->[\Delta] 2CuO}$ 表面变黑

​ $\ce{3Fe+2O2 ->[点燃] Fe3O4}$

  • Au 在高温下也不与O2反应 说明金化学性质稳定，金耐腐蚀

总结：

  • 活动性： Al, Zn, Mg > Fe, Cu > Au

  • 判断依据：金属能否与氧气反应，及反应难易程度

二、能与酸反应

  • $\ce{Mg + 2HCl -> MgCl2 + H2 ^}$
  • $\ce{Zn + 2HCl -> ZnCl2 + H2 ^}$
  • $\ce{Fe + 2HCl -> FeCl2 + H2 ^}$
  • $\ce{Cu + 2HCl}$ 不反应
  • $\ce{Mg + H2SO4 -> MgSO4 + H2 ^}$
  • $\ce{Zn + H2SO4 -> ZnSO4 + H2 ^}$
  • $\ce{Fe + H2SO4 -> ZnSO4 + H2 ^}$

现象：Mg Zn有气泡产生；Fe有气泡产生，溶液由无色变为浅绿色 （Cu2+ 蓝色，Fe2+ 浅绿色）

活动性：Mg > Zn > Fe > Cu

判断依据：金属能否与酸反应，及反应的剧烈程度（快慢）

置换反应：一种单质和一种化合物反应，生成另一种单质和另一种化合物的反应

​ $\ce{A + BC -> B + AC}$

三、金属能与盐溶液反应

$\ce{Fe + CuSO4 -> FeSO4 + H2 ^}$ （湿法炼铜，不用铁桶装波尔多液，曾青得铁化为铜）

$\ce{2Al + 3CuSO4 -> 3Cu + Al2(SO4)3}$

  • 现象：Al表面有红色物质析出，溶液由蓝色变为无色
  • 表示Al活动性比Cu强

$\ce{Cu + 2AgNO3 -> 2Ag + Cu(NO3)2}$

  • $\ce{AgNO3}$溶液存储在棕色细口瓶：$\ce{AgNO3}$见光易分解
  • 现象：Cu表面有银白色物质析出，溶液由无色变为蓝色
  • Ag比Cu活泼

$\ce{Cu + Al2(SO4)3}$ 不反应

活动性： Al > Cu > Ag 依据：金属能否与盐反应

四、金属活动性顺序

**K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au **

钾钙钠镁铝 锌铁锡铅氢 铜汞银铂金

应用：

  • 在金属活动性顺序里，金属位置越靠前活动性越强
  • 在金属活动性顺序里，位于氢前面的金属能置换出盐酸、稀硫酸中的氢
  • 在金属活动性顺序里，位于前面的金属能把位于后面的金属从它们的盐溶液中置换出来

比较三种金属活动性，Fe Cu Ag 为例

  • 两金加一盐 $\ce{Fe}$, $\ce{Ag}$, $\ce{CuSO4}$溶液
  • 两盐加一金 $\ce{Cu}$, $\ce{FeSO4}$溶液,$\ce{AgNO3}$溶液

等酸足金：生成H2质量相等

等金足酸：生成H2质量和金属的相对原子质量成反比
